# 🚀 AI Interview Platform

An AI-powered full-stack web application built using the **MERN stack** that helps students practice interviews and allows recruiters to evaluate candidates efficiently.

---

## ✨ Features

### 👨‍🎓 Student Features
- AI-based mock interview system  
- Personalized interview questions  
- Performance scoring & feedback  
- Resume upload functionality  
- Dashboard to track progress  

### 🧑‍💼 Recruiter Features
- Create and manage interview sessions  
- Evaluate candidates  
- View candidate reports  
- Manage interview questions  

---

## ⚙️ Tech Stack

### Frontend
- React.js  
- CSS / Tailwind (if used)  
- Axios  

### Backend

- Express.js  
- MongoDB
